# MVP Plan (Round 1)

## Goal

Build a stable base that can be extended in later rounds without structural rewrites.

## Milestones

1. Scaffold directories and configuration files.
2. Run FastAPI app with registered routes.
3. Initialize SQLite schema via script.
4. Define placeholders for models, schemas, and services.

## Acceptance

- service starts
- health endpoint responds
- DB tables are creatable
- API docs render
